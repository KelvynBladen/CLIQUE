# CLIQUE 1.0.0

## New features

* Initial CRAN submission.
* Implements CLIQUE (Conditional Local Importance by Quantile Expectation).
* Supports regression, binary classification, and multi-class classification.
* Supports univariate and multivariate regression.
* Includes cross-validated (`clique()`) and pretrained-model (`clique_eval()`) workflows.
* Supports custom loss functions.
* Includes PDP visualizations for multi-class classification tasks.
